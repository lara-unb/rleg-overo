RLeg - Overo

This is a software to run in a Gumstix - Overo based embbed system developed by @caiogustavo for control and instrumentation of the fist robotic transtibial and transfemoral prostheses.

Requirements

1. Arm Cross Compiler

2. Doxygen

Folders

  • doc : Folder to store the documentation generated by Doxygen
  • Matlab : Datalogger files to export to Matlab
  • src : Source Code

Reference:

  • ANGELO, CAIO GUSTAVO MESQUITA, Desenvolvimento do sistema embarcado de uma prótese robótica para amputados transfemurais
